Laravel widget https://github.com/arrilot/laravel-widgets

Features : Asynchronous widgets, reloadable widgets, console generator, caching

Use : insert in a view an other view (fragment) with a dedicated controller (widget class)


Pro : 	- easy to implement ! view, 1 controller (widget class) , @widget('recentNews')      	- possibility to add parameter @widget('recentNews', ['count' => 10])
	- default or custom namespace
	- Asynchronous widgets : load widget content with ajax using @asyncWidget, Widget params are encrypted
	- Reloadable widget : automatically reload widget every N seconds :  public $reloadTimeout = 10;
	- Container : Async and Reloadable widgets both require some DOM interaction, they wrap all widget output in a html container.  AbstractWidget::container()
	- Caching, public $cacheTime = 60;
	- Widget group : pWidget::group('sidebar')->position(5)->addWidget('widgetName1', $config1);
Widget::group('sidebar')->position(4)->addAsyncWidget('widgetName2', $config2);
{{ Widget::group('sidebar')->display() }}
